# Анализ рейтингов фильмов: Fandango vs другие платформы
Описание проекта
Сравнительный анализ рейтингов фильмов с платформы Fandango с рейтингами других агрегаторов (Rotten Tomatoes, Metacritic, IMDB) для выявления системных расхождений в оценках.

## Цели исследования
- Выявить различия между отображаемыми звездами (STARS) и фактическими рейтингами (RATING) на Fandango
- Сравнить распределения оценок между разными платформами
- Проанализировать согласованность оценок критиков и пользователей
- Обнаружить фильмы с наибольшими расхождениями в оценках

## Ключевые этапы анализа
1. Исследование данных Fandango
- Анализ 504 фильмов с оценками от 0 до 5 звезд
- Обнаружено: 69 фильмов имеют 0 голосов
- Выявлена сильная корреляция между STARS и RATING (0.99)
- Создана колонка YEAR для анализа по годам

2. Визуализация расхождений
- KDE plot показал систематическое завышение отображаемых звезд (STARS) над фактическим рейтингом (RATING)
- Наибольшее расхождение: фильм "Turbo Kid" - 5 звезд при реальном рейтинге 4.0

3. Сравнение с другими платформами
- Объединение данных с Rotten Tomatoes, Metacritic, IMDB (145 общих фильмов)
- Нормализация всех рейтингов к шкале 0-5 для сравнения
- Clustermap анализ выявил кластеры оценок

4. Ключевые находки
- Fandango consistently inflates ratings - оценки систематически завышены
- Rotten Tomatoes critics vs users - средняя абсолютная разница 15.1%
- Worst films analysis - даже плохо оцененные фильмы получают высокие оценки на Fandango

## Техническая реализация
Библиотеки: Pandas, NumPy, Matplotlib, Seaborn, Jupyter

Методы:
- EDA и визуализация распределений
- Слияние и нормализация данных
- Корреляционный анализ
- Кластерный анализ (clustermap)

## Результаты и инсайты
- Fandango показывает завышенные оценки - KDE распределения смещены вправо
- Низкая согласованность между критиками и пользователями на Rotten Tomatoes

### Топ-5 фильмов с наибольшими расхождениями:
- Пользователи любят, критики ненавидят: "Do You Believe?" (-66)
- Критики любят, пользователи ненавидят: "Mr. Turner" (+42)
- Нормализованные рейтинги показывают систематическое смещение Fandango
